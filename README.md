# My Arch + Hyprland Dotfiles

Tokyo Night themed Hyprland setup on Arch Linux.

## Features
- Hyprland window manager with UWSM
- Waybar with custom styling
- Kitty terminal with Tokyo Night theme
- Nordic GTK theme
- Wofi launcher
- Dunst notifications
- Clipboard history (cliphist)
- Screenshot tools (grim + slurp)

## Installation

1. Clone the repo:
```bash
git clone https://github.com/yourusername/dotfiles.git
cd dotfiles
```

2. Run the install script:
```bash
./install.sh
```

3. Add wallpapers to `~/Pictures/Wallpapers/`

4. Update `~/.config/hypr/hyprpaper.conf` with your wallpaper paths

5. Log out and back in

## Keybinds
- `Super + Shift + S` - Screenshot selection
- `Super + Shift + C` - Clipboard history
- Power button in Waybar - Logout/shutdown menu
